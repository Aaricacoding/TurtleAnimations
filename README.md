## вңҰ What Is TurtleAnimations?

**TurtleAnimations** is a curated gallery of generative art and geometric animations, crafted entirely with Python's built-in `turtle` library. Each script is a small mathematical poem вҖ” a set of equations that bloom into mesmerizing visual patterns on your screen.

This project exists to prove one thing:

> **You don't need a game engine, GPU renderer, or complex framework to create beautiful art. You just need math, code, and curiosity.**

Whether you're a student learning Python, a developer exploring creative coding, or an artist fascinated by algorithms вҖ” there's something here for you.

## рҹҡҖ Getting Started

### Prerequisites

All you need is Python. That's it.

```bash
# Check your Python version (3.6+ recommended)
python --version
```

The `turtle` module ships with every standard Python installation вҖ” no `pip install` required.

### Installation

```bash
# 1. Clone the repository
git clone https://github.com/your-username/TurtleAnimations.git

# 2. Navigate into the project
cd TurtleAnimations

# 3. Run any animation вҖ” it's that simple!
python parametric_heart.py
```

### Running Each Animation

```bash
# рҹ’– Watch the heart take shape
python parametric_heart.py

# рҹҢҖ Watch arcs weave together
python arc_weave.py

# рҹҢҲ Spin up the rainbow wheel
python rainbow_geometric_wheel.py

# вҳҖпёҸ Let the sunburst spiral unfold
python sunburst_spiral.py
```

> рҹ’Ў **Tip:** Each animation opens in its own Turtle window. Close the window to exit.
